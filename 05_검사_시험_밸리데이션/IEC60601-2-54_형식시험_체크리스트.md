## 1. 목적
IEC 60601-2-54:2022 (Medical electrical equipment — Part 2-54: Particular requirements for the basic safety and essential performance of X-ray equipment for radiography and radioscopy) 에 따른 형식시험(Type Test) 항목을 체크리스트로 정리하여, 설계 완료 후 공인시험소 형식시험 준비 및 자체 사전 점검에 활용한다.

## 2. 적용 규격
- **주 규격**: IEC 60601-2-54:2022 (Ed. 2.0)
- **일반 규격**: IEC 60601-1:2005/AMD1:2012/AMD2:2020
- **방사선 방호**: IEC 60601-1-3:2008/AMD1:2013/AMD2:2021
- **EMC**: IEC 60601-1-2:2014/AMD1:2020
- **사용적합성**: IEC 62366-1:2015/AMD1:2020
- **SW 수명주기**: IEC 62304:2006/AMD1:2015
- **위험관리**: ISO 14971:2019
- **Detector 성능**: IEC 62220-1-1:2015 (DQE, MTF, NPS)
- **DAP**: IEC 60580:2019
- **TRF**: IECEE TRF 60601-2-54I:2023

## 4. 투시(Fluoroscopy) 모드 QC 프로토콜

### 4.1 목적
X-ray 장비의 투시 기능에 특화된 QC 항목을 정의하여, 촬영(Radiography) 모드 QC(§8, 기존 문서)와 구분하여 관리한다.

### 4.2 적용 규격
- IEC 60601-2-54:2022 §203.8 (Radioscopy 요구사항)
- IEC 60601-2-43:2022 (투시전용장비 참조, 해당 시)
- AAPM TG-272 (Comprehensive Acceptance Testing of Fluoroscopy Imaging Systems, 2022)
- EFOMP Quality Control of Dynamic X-ray Imaging (2024)
- 진단용 방사선 안전관리규칙(제1122호) — 투시장비 검사항목

### 4.3 투시 QC 항목

| QC 항목 | 시험 방법 | 판정 기준 | 주기 | 주체 |
|---------|----------|-----------|------|------|
| 입사면 선량률 (Normal) | PMMA 팬텀, Dosimeter 측정 | ≤88 mGy/min | 출하/설치/정기 | QC/검사기관 |
| 입사면 선량률 (High-level) | PMMA 팬텀, Dosimeter 측정 | ≤176 mGy/min | 출하/설치/정기 | QC/검사기관 |
| ABC 응답 (밝기 안정성) | Cu 필터 추가·제거 시 영상밝기 변동 | 밝기 변동 ≤±20% | 출하/정기 | QC |
| 저대비 해상도 (투시) | 저대비 팬텀 투시 모드 | 가시 디테일 ≥설계 목표 | 출하/정기 | QC |
| 공간 해상도 (투시) | Line-pair 팬텀 투시 모드 | ≥설계 목표 lp/mm | 출하/정기 | QC |
| 시간 해상도 (Temporal) | 회전 팬텀 또는 펄스 주파수 확인 | 설정 fps ±10% | 출하 | QC |
| DAP 누적 정확도 | 기준선량계 vs DAP 비교 (투시 10분) | ±35% (IEC 60601-2-43) | 출하/정기 | QC |
| 선량 경보 작동 | 사전설정 임계값 도달 시 경보 확인 | 임계값 ±10% 이내 경보 | 출하/설치 | QC |
| 5분 타이머 경보 | 투시 5분 연속 시 알림 | 5분 ±15초 이내 | 출하/설치 | QC |
| Last-Image-Hold | 투시 종료 후 마지막 프레임 유지 | 프레임 유지 확인 | 출하/설치 | QC |
| 투시→촬영 전환 | 모드 전환 후 kVp/mAs 정확성 | 촬영 모드 기준 준용 | 출하 | QC |
| Grid 유무별 선량률 | Grid 장착/미장착 시 비교 | 미장착 시 선량률 적절 감소 | 출하 | QC |

### 4.4 투시 QC 단계별 적용

| 단계 | 적용 항목 |
|------|-----------|
| 형식시험 | 전항목 (공인시험소) |
| 출하시험 | 선량률, ABC, 해상도, DAP, 경보, LIH, 모드전환, Grid |
| 설치검사 | 선량률, ABC, 경보, 5분타이머, LIH + 환경 검증 |
| 정기검사 (3년) | 선량률, ABC, 해상도, DAP |
| 일상 QC (주간) | ABC 안정성 (간이 팬텀), 5분 타이머 확인 |

### 4.5 PMS 환류
- 투시 영상품질 클레임 코드: FL-001(선량률 이상), FL-002(ABC 불안정), FL-003(해상도 저하), FL-004(경보 미작동)
- CAPA 트리거: 동일 코드 2건/분기 시 경향분석 착수
- `08_시판후_감시_PMS/PMS_개요.md` §3 데이터원천에 투시 QC 코드 추가 필요
