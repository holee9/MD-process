# 위험관리 - ISO 14971:2019

## 1. 표준 상태
- 현행판: ISO 14971:2019 (3판)
- 유럽 조화판: BS EN ISO 14971:2019+A11:2021 (EU MDR/IVDR과 조화, Annex ZA/ZB)
- 지침: ISO/TR 24971:2020 (적용 가이던스)

## 2. 위험관리 프로세스 (수명주기 전반)
1. 위험관리 계획 수립
2. 위험 분석 (intended use, 위해요인 식별)
3. 위험 평가 (가능성 × 심각성)
4. 위험 통제 (본질안전·보호수단·정보제공)
5. 잔여위험 평가 및 편익-위험 분석
6. 생산·시판후 정보 수집 및 위험 재평가

## 3. 고려할 위험 유형
- 생체적합성, 전기·기계, 방사선, 소음
- 데이터·시스템 보안(사이버보안)
- 사용성(use error)
- 정보보안·프라이버시

## 4. QMS 및 규제와의 연결
- FDA QMSR: 위험관리를 QMS 전반에 명시적 요구
- EU MDR: GSPR Annex I 충족 증빙의 핵심
- MFDS GMP: 별표2 전반에 위험기반 접근 반영
- ISO 13485 조항 7.1, 7.3.3, 7.4.1, 8.2.1 등과 연동

## 5. 위험관리 파일(RMF) 기본 구성
- 위험관리 계획서
- 위해요인·위험 목록
- 위험 통제 조치 내역
- 생산·시판후 위험 추적 로그
- 위험관리 보고서

## 6. 실무 체크포인트
- 위험 수용 기준의 객관성·조직 정책 문서화
- 사용성 평가(IEC 62366-1)와 위험분석 연결
- 소프트웨어 포함 기기의 경우 IEC 62304 위험관리와 연결
- 사이버보안: FDA Premarket Cybersecurity Guidance, IEC 81001-5-1 연계
