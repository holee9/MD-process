# 진단용 방사선 발생장치의 안전관리에 관한 규칙 (보건복지부령 제1122호)

## 1. 개요
- 법령명: 진단용 방사선 발생장치의 안전관리에 관한 규칙
- 근거: 의료법 제37조(진단용 방사선 발생장치)
- 최신 개정: 2025-07-18 (보건복지부령 제1122호)
- 시행일: 2025-07-18

## 2. 적용 범위
의료기관이 보유·운용하는 진단용 방사선 발생장치(X선 장치, CT, 치과용 X선, 투시장치 등)의 안전관리에 적용

## 3. 주요 조항 요약

### 3.1 설치·신고 (제3조~제4조)
- 의료기관 개설자는 진단용 방사선 발생장치를 설치한 때 관할 시장·군수·구청장에게 신고
- 장치 변경(교체·이전·추가) 시에도 신고 의무

### 3.2 검사 (제5조~제8조)
- **정기검사**: 설치 후 3년마다 성능검사 (안전관리책임기관 또는 지정검사기관 시행)
- **수시검사**: 중대 결함 발견 또는 사고 발생 시
- 검사 항목: 관전압 정확도, 선량 재현성, 반가층(HVL), 누설선량, 산란선 방호, 영상품질 등
- 부적합 판정 시 사용 중지 → 보수 후 재검사 합격 시 사용 재개

### 3.3 안전관리책임자 (제9조)
- 의료기관은 방사선 안전관리책임자를 선임(방사선사 또는 관련 자격자)
- 책임: 장치 관리, 종사자 피폭관리, 방호시설 점검

### 3.4 방어시설 기준 (제10조, 별표)
- 방사선 관계구역 설정, 차폐벽·방호문 기준
- X선 장치실 차폐 기준: 1차 방어벽 ≥1.5mm Pb 등가, 2차 방어벽 ≥ 관련 기준

### 3.5 피폭관리 (제11조~제12조)
- 방사선 관계 종사자 개인선량계(TLD/OSL) 착용 의무
- 연간 유효선량 한도: 50mSv/년 및 5년간 100mSv (의료법 시행규칙 준용)
- 피폭 기록 보존: 5년

### 3.6 환자 방호 (제13조)
- 불필요한 피폭 최소화 원칙(ALARA)
- 생식선 방호 등 환자 방호 조치

## 4. X-ray 시스템 제조사 관련 시사점
| 항목 | 제조사 대응 필요 사항 |
|------|----------------------|
| 관전압·선량 정확도 | 출하 시 규격 내 교정 확인, 사용자 매뉴얼에 검사 주기 명기 |
| 누설선량 | 형식시험(Type Test) 시 IEC 60601-1-3 기준 충족 증빙 |
| 영상품질 | 성능검사 항목과 자사 QC 프로토콜 매핑 |
| 차폐 설계지원 | 설치 가이드에 차폐 계산 기초 데이터(Workload, Use Factor) 제공 |
| ALARA 지원 기능 | DAP(Dose Area Product) 표시, AEC, 자동 콜리메이션 등 |
