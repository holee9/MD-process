## 2. 세부 교차검증

### 2.1 선량 관련 SW 안전성 클래스
- X-ray 매핑표 §5: AEC·노출제어·차폐 인터록 SW = Class C
- IEC 62304 문서(기존): Class C 확정과 일치
- ISO 14971 문서(기존): 과노출 Hazard 정의 — 일관
- 교차: 판정기준 및 심각도 정의 내부 모순 없음

### 2.2 사이버보안 ↔ 임상 안전 이중 판정
- 81001-5-1 문서 §6: 취약점 발생 시 ISO 14971 재평가, CVSS v3.1 + 의료기기 Rubric
- 매핑표 §4.2: FDA Part 1020과 cybersecurity 모두 제출 의무
- 결론: 단일 Change Control Board에서 선량 영향과 보안 영향을 동시 판정해야 함 → 03/05/07 문서 간 ChangeControl SOP 초안 필요(신규 이슈 필요)

### 2.3 EU NB 심사 필요 자료 정합
- 매핑표 §4.1: 60601-2-54 형식시험, 62304 SDLC, 81001-5-1 Capability 보고서
- 81001-5-1 문서 §3 Plan 11개 절: NB 제출 시 단일 Security Management Plan이 MDR TD에 직접 참조 가능
- 권고: TD 목차 템플릿에 "Cybersecurity Annex" 단일 항목으로 추가 (TD 템플릿 개정 필요 → 005_04_GSPR 체크리스트와 연계)

### 2.4 MFDS 제출 정합
- 매핑표 §4.3: 공통기준규격·방사선 안전관리 기준규격
- 81001-5-1 문서 §2: MFDS 사이버보안 가이드(2022)
- 디지털의료제품법(2025-01-24) 시행규칙 별도 제출물 존재 가능 — **오픈 이슈**

신규 2개 문서는 기존 62304/14971/GSPR 문서와 내부 모순 없이 정합하며, 본 프로젝트의 X-ray 특화 규제 커버리지를 보완한다.
