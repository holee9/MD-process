# SOP-PMS-001 불만처리 및 부작용보고 절차 — v0.2

> **문서번호**: SOP-PMS-001 | **버전**: v0.2 (R.0) | **시행일**: 2026-05-03
> **작성**: PMS 담당 | **검토**: QA | **승인**: QA장

---

## 1. 목적

고객 불만(Complaint) 접수부터 조사·판정·규제 당국 보고·종결까지의 절차를 정하고, 부작용(Adverse Event) 발생 시 보고 기한·양식·판정 기준을 명확히 한다.

## 2. 적용 범위

- 자사 제조 의료용 X-ray 시스템, 디지털 디텍터(FPD), 영상처리 SW(AI 포함)
- 전 시장(국내 MFDS, 미국 FDA, EU/NB) 출하 제품

## 3. 관련 규제·표준

| 규제/표준 | 조항 | 요구사항 |
|-----------|------|----------|
| ISO 13485:2016 | 8.2.1, 8.2.2, 8.2.3 | 피드백, 불만처리, 규제 당국 보고 |
| FDA 21 CFR 803 | 전문 | MDR(Medical Device Reporting) |
| FDA QMSR (21 CFR 820) | ISO 13485 참조편입 | 불만 파일 유지 |
| EU MDR 2017/745 | Art. 87–92 | Vigilance 보고 |
| MEDDEV 2.12/1 rev.8 | 전문 | Vigilance 가이드 (MDR 전환기 참조) |
| 의료기기법 | 제31조 | 부작용 보고 의무 |
| 의료기기법 시행규칙 | 별표 17 | 부작용 보고서 양식 |

## 4. 용어 정의

| 용어 | 정의 |
|------|------|
| **불만(Complaint)** | 시판된 기기의 성능·안전·품질에 대한 외부 의사소통으로, 부적합 가능성을 내포하는 것 |
| **부작용(Adverse Event)** | 기기의 사용과 관련하여 환자·사용자·제3자에게 발생한 사망·건강 악화 또는 그 위협 |
| **FSCA** | Field Safety Corrective Action — 현장 시정조치 |
| **FSN** | Field Safety Notice — 현장 안전 공지 |
| **중대 부작용(Serious AE)** | 사망, 영구 기능장애, 입원/입원 연장, 생명 위협, 의학적 중재 필요 |

## 5. 책임과 권한

| 역할 | 책임 |
|------|------|
| **QA/불만처리 담당** | 불만 접수·초기 평가·조사 조정·기록 관리 |
| **RA** | 규제 보고 판정·보고서 작성·제출 |
| **기술팀(HW/SW/AI)** | 원인 조사·기술 분석 |
| **서비스팀(FSE)** | 현장 정보 수집·현장 시정조치 이행 |
| **QA장** | 최종 판정 승인, FSCA/FSN 발행 승인 |
| **대표이사** | 리콜 결정 승인 |

## 6. 절차

### 6.1 불만 접수 (D+0)

```
입력: 고객 전화/이메일, 대리점 보고, FSE 보고, 규제DB 경보, 내부 발견
활동:
 1. QA가 접수 → 접수번호(CPL-YYYY-NNN) 부여
 2. F-PMS-001(불만·부작용 초기 평가서) 작성 개시
 3. 불만 유형 분류:
    □ HW 고장  □ SW 오작동  □ 방사선 관련  □ 전기안전
    □ AI 성능  □ 사이버보안  □ 사용오류    □ 기타
출력: 접수된 F-PMS-001 (접수일·경로·제품정보 기재)
판정: 접수 완료 확인 (24시간 이내)
```

### 6.2 초기 위험 평가 (D+1~3)

```
입력: F-PMS-001 접수 정보, 제품 이력(DHR)
활동:
 1. 환자·사용자 위해 여부 판정
 2. 심각도(Severity) × 발생 확률 → 위험 수준 결정
    (07_위험관리/위험관리_개요 §위험평가 매트릭스 적용)
 3. 규제 보고 필요 여부 판정:
    - 중대 부작용 → 즉시 보고 절차 개시 (§6.4)
    - 비중대 → 조사 진행 (§6.3)
 4. 즉각 조치 필요 여부 결정 (사용 중지 권고, 회수 등)
출력: F-PMS-001 초기 평가 섹션 완료
판정: QA장 승인
```

**X-ray 특화 판정 기준:**

| 불만 유형 | 심각도 기본 등급 | 즉각 조치 |
|-----------|-----------------|-----------|
| X-ray 튜브 파열/방전 | S4(치명) | 즉시 사용 중지, 방사선 피폭 평가 |
| 선량 표시 오류(SW) | S3(중대) | SW 패치 긴급 배포 검토 |
| FPD 화소 불량 다발 | S2(경미)~S3 | 경향 분석, 임계치 초과 시 교체 |
| 콜리메이터 오작동 | S4(치명) | 즉시 사용 중지 |
| AI 오진율 이상 상승 | S3(중대) | 모델 비활성화 또는 경고 표시 |
| 사이버보안 침해 | S3~S4 | 네트워크 격리, 증거 보전 |

### 6.3 조사 (D+3~30)

```
입력: 초기 평가 결과, 불만 기기(가능 시), 관련 기록
활동:
 1. 기술팀 원인 조사 (고장 분석, SW 로그, 재현 시험)
 2. 유사 불만 이력 조회 (동일 모델·로트·부품)
 3. 근본원인 분석 (5-Why, 피쉬본, FTA 등)
 4. 경향 분석 (해당 불만 유형 누적 건수 추이)
출력: 조사 보고서 (F-PMS-002)
판정: 근본원인 식별 및 시정조치 필요 여부 결정
       CAPA 필요 시 → CAPA 절차 연계
```

### 6.4 규제 당국 보고

#### 6.4.1 보고 기한

| 지역 | 보고 유형 | 기한 | 양식/시스템 |
|------|-----------|------|------------|
| **EU** | 중대 AE (사망·심각 위협) | **2일** | EUDAMED Vigilance |
| **EU** | 기타 중대 AE | **15일** | EUDAMED Vigilance |
| **EU** | 경향 보고 | 인지 후 즉시 | EUDAMED |
| **US** | 사망·중상 | **5 work days** (인지 후) | FDA eMDR (MedWatch 3500A) |
| **US** | 기타 보고 대상 | **30 calendar days** | FDA eMDR |
| **KR** | 사망·중대위해 | **7일** | 의약품안전나라 |
| **KR** | 중대 부작용 | **15일** | 의약품안전나라 |

#### 6.4.2 보고서 작성

```
입력: 조사 결과, 초기 평가서
활동:
 1. RA가 해당 지역 규제 양식에 맞춰 보고서 작성
 2. QA장 검토·승인
 3. 기한 내 제출 (전자시스템 또는 서면)
 4. 후속보고(Follow-up Report) 필요 시 조사 완료 후 제출
출력: 제출된 규제 보고서, 접수 확인
판정: 기한 준수 확인
```

### 6.5 시정조치 (FSCA/FSN)

```
입력: 조사 결과, 위험 평가
활동:
 1. FSCA 필요 여부 판정 (수용 불가 위험 존재 시)
 2. FSCA 유형 결정:
    - 리콜(회수), SW 업데이트, 라벨링 변경, 사용 제한 안내
 3. FSN 작성·배포 (고객·규제당국·유통자)
 4. FSCA 이행 추적 (이행률 모니터링)
 5. 변경통제(SOP-CC-001) 연계 (설계변경 수반 시)
출력: FSCA 기록, FSN 문서, 이행 추적 기록
판정: QA장(FSCA) / 대표이사(리콜) 승인
```

### 6.6 종결

```
입력: 조사 완료, 시정조치 이행 확인
활동:
 1. 모든 조치 완료 확인
 2. 불만 파일 종결 (F-PMS-001에 종결일·결과 기재)
 3. PMS 데이터에 반영 (PSUR 집계용)
 4. CAPA 효과성 확인 (해당 시)
출력: 종결된 불만 파일
판정: QA장 종결 승인
```
