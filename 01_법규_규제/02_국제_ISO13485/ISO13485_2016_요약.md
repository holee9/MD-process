# ISO 13485:2016 - Medical devices QMS

## 1. 표준 상태 (2026-04 기준)
- 현행판: ISO 13485:2016 (3판)
- 2025.1.15 ISO/TC 210 시스템 검토 → reconfirmed (현행 유지)
- 차기 개정 시 검토 예정: 사이버보안, AI/ML, PMS 강화

## 2. 표준의 구조
| 조항 | 내용 |
|------|------|
| 4 | 품질경영시스템 |
| 5 | 경영책임 |
| 6 | 자원관리 |
| 7 | 제품실현 (설계개발·구매·생산·모니터링) |
| 8 | 측정·분석·개선 |

## 3. 2016판의 ISO 9001 대비 강화 요소
- 위험기반 접근 (risk-based approach)을 전 프로세스에 명시
- 규제요구사항 준수를 조직의 책임으로 명확화
- 클린룸 등 작업환경 관리 요구 강화
- 소프트웨어 밸리데이션 요구 (QMS 및 제품 모두)
- 구매·공급자 통제 강화 (위험 비례)
- 불만처리·사후 감시·보고 요구 강화

## 4. 규제 간 채택 현황
- 대한민국 GMP 별표2: ISO 13485 정합
- FDA QMSR (2026.2.2): ISO 13485:2016을 참조 편입
- EU MDR/IVDR: 조화 표준으로 인정
- MDSAP: ISO 13485 기반 단일 감사

## 5. 실무 체크포인트
- 문서체계가 ISO 13485 조항 번호와 매핑되는지 확인
- 위험관리(ISO 14971)와 QMS의 연결 고리 문서화
- 공급자 관리에 위험도 기반 차등 적용
