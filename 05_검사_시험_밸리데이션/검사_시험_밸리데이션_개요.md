# 검사·시험·밸리데이션 개요

## 1. 목적
수입·공정·완제품 단계별 검사와 시험 체계, 그리고 공정·소프트웨어·시험방법 밸리데이션 체계를 확립하여 제품이 규정 요구사항과 의도된 사용 목적에 적합함을 보장한다.

## 2. 관련 규제 근거

## 3. 검사 체계

### 3.1 수입검사(Incoming)
- 구매품 사양서·검사기준서 기반 샘플링
- 공급자 등급에 따른 검사 수준 차등(면제·간이·정밀)
- 부적합 시 제3자 검교정·공급자 시정 연계

### 3.2 공정검사(In-Process)
- 핵심공정(CTQ) 식별 → 관리계획(Control Plan) 운영
- 관리한계 이탈 시 일시 정지·근본원인분석(RCA)

### 3.3 최종검사(Finished)
- 출하 기준 전수/샘플링 구분, 멸균·성능시험 포함
- 제품출하 승인권자(QA) 서명 후에만 시장 출하

## 4. 밸리데이션 체계

### 4.1 공정 밸리데이션 (Process Validation)
- 대상 판단: 결과를 후속 검사로 완전 검증 불가능한 공정 (예: 멸균, 용접, 접착, 세척, 포장 실링)
- 단계: DQ → IQ → OQ → PQ
- 재밸리데이션 트리거: 설계 변경, 장비 이설/교체, 환경 변경, 연간 주기 검토

### 4.2 시험방법 밸리데이션 (Test Method Validation / MSA)
- 특성치별: Gage R&R, 정확도, 정밀도, 선형성, 안정성
- 파괴시험·계수형 자료는 Attribute MSA 적용

### 4.3 소프트웨어 밸리데이션
- 제품 SW: IEC 62304 기반 클래스별(A/B/C) 수명주기 관리
- 생산·QMS SW: 21 CFR 820.70(i) / ISO/TR 80002-2, GAMP 5 접근
- AI/ML 활용 SW: FDA PCCP(Predetermined Change Control Plan), EU MDR + AI Act 고려

### 4.4 멸균 밸리데이션
- EO: ISO 11135 / 방사선: ISO 11137 / 습열: ISO 17665 / 무균공정: ISO 13408

## 5. 기록 요건
- 검사성적서, IQ/OQ/PQ 프로토콜·보고서, 변경이력, 재밸리데이션 근거
- FDA QMSR 하 § 820.180(c) 예외 폐지로 내부감사·경영검토·공급자감사 기록을 외부열람 대비 품질 상향 유지
