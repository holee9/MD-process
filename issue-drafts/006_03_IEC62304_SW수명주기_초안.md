## 배경
X-ray 시스템 제어 SW, detector 펌웨어, 영상처리·판독 보조 SaMD 개발에 적용되는 IEC 62304:2006/A1:2015/A2:2020 의 수명주기 요구를 단일 문서로 정리. SW 안전성 클래스(A/B/C) 결정, SOUP 관리, 사이버보안(IEC 81001-5-1) 오버레이, AI/ML 특화 고려사항을 포함.

## 체크리스트
- [x] 표준 현행 상태 확인 (A2:2020 Legacy 조항, OJEU 미등재 상태)
- [x] 제품 유형별 적용 범위 매핑 (SiMD / SaMD / SOUP / 비제품 SW)
- [x] SW 안전성 클래스 결정 기준 및 X-ray 제품 예시
- [x] §5–§9 프로세스 단계별 요구 정리
- [x] SOUP 관리 실무 지침
- [x] IEC 81001-5-1 사이버보안 오버레이
- [x] AI/ML SW 특화 고려 (IMDRF N41, FDA GMLP, AI Act Annex IV)
- [x] DHF 산출물 목록
- [ ] SOUP 평가 → ISO 14971 RMF 자동 연동 양식 초안
