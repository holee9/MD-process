## 확인된 이슈

1. **index.lock 잔존** — `.git/index.lock`(0 byte)이 남아 있어 `git add/commit/reset` 등 인덱스 쓰기 작업이 전부 실패.
2. **index 파일 손상** — 원본 `.git/index`가 0 byte로 잘렸으나 권한 이슈로 교체 불가.
3. **인증 토큰 부재** — `.secrets/token.txt`가 비어 있어 자동 push / issue 등록 불가.
4. **working tree 수정 사항** — `.gitignore`, `00_프로젝트관리/색인_INDEX.md` 두 파일이 last commit 대비 변경됨(미커밋).

- Windows 측에서 `의료기기 제조 업무규칙 개발\.git\index.lock` 및 `index` 파일 수동 삭제 후 `git reset` 1회 실행.
