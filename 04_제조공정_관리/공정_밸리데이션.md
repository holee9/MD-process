# 제조공정 관리 및 밸리데이션

## 1. 근거
- ISO 13485 7.5.6 (공정 밸리데이션)
- MFDS GMP 별표2 7.5
- FDA Process Validation Guidance (2011 및 FDA QMSR 편입)

## 2. 밸리데이션 대상 공정
- 결과를 완제품 검사만으로 검증할 수 없는 공정
  - 예) 멸균, 용접, 사출, 접합, 클린룸 조립, 소프트웨어 적용 공정

## 3. 밸리데이션 단계 (IQ-OQ-PQ)
| 단계 | 목적 |
|------|------|
| IQ (설치적격성) | 장비·설비가 사양대로 설치 |
| OQ (운전적격성) | 지정 조건에서 올바르게 작동 |
| PQ (성능적격성) | 실제 생산 조건에서 반복 성능 달성 |

## 4. 소프트웨어 밸리데이션
- 생산·QMS에 사용하는 S/W도 밸리데이션 대상
- IEC 62304 (의료기기 S/W 수명주기), GAMP 5 참조
- 위험기반 접근 (risk-based validation)

## 5. 재밸리데이션 트리거
- 장비·공정·재료·S/W의 중대한 변경
- 부적합 경향 발생
- 주기적(사업장 규정에 따른) 재검토

## 6. 실무 체크포인트
- 각 공정의 중요 공정 파라미터(CPP)·중요 품질 특성(CQA) 정의
- 통계적 표본설계 근거 문서화
- 멸균은 ISO 11135/11137/17665 등 해당 표준 준수
