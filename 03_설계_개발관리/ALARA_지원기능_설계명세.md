# ALARA 지원 기능 설계 명세 (v0.1)

## 1. 목적
진단용 X-ray 시스템의 ALARA(As Low As Reasonably Achievable) 원칙 구현을 위해 제품에 탑재할 선량 최적화 기능의 설계 요구사항을 정의한다.

## 2. 근거 규격
| 규격 | 관련 조항 | 요구 내용 |
|------|-----------|-----------|
| 진단용 방사선 안전관리규칙 제1122호 | 제13조 (환자 방호) | 불필요한 피폭 최소화(ALARA) |
| IEC 60601-1-3:2008+A2:2021 | §7 (방사선 방호) | 사용자·환자 선량 최소화 설계 원리 |
| IEC 60601-2-54 | §203.7.8 (AEC) | 자동노출제어 성능 요구 |
| IEC 60580:2019 | 전체 | DAP meter 정확도·표시 요구 |
| 21 CFR 1020.31 | (d) AEC | 미국 FDA 방사선 성능 표준 |
| EU MDR Annex I | §14.2(c), §16.2 | 진단기기 선량 최적화 요구 |

## 3. ALARA 지원 기능 목록

### 3.1 DAP(Dose Area Product) 실시간 표시
- **설계 요구**: 촬영/투시 중 누적 DAP 값 UI에 실시간 표시
- **정확도**: 기준선량계 대비 ±25% 이내 (IEC 60580 §5)
- **단위**: mGy·cm² (SI), 사용자 설정으로 μGy·m² 전환 가능
- **기록**: 촬영 건별 DAP 값 DICOM Radiation Dose SR (Structured Report)로 자동 저장
- **경고**: 프로토콜별 DRL(Diagnostic Reference Level) 초과 시 시각·음성 경고

### 3.2 AEC(Automatic Exposure Control)
- **설계 요구**: 피사체 두께·밀도에 따라 mAs 자동 조절
- **성능**: 두께 변화(10~30cm PMMA) 시 영상 수용체 입사 선량 변동 ±25% 이내 (IEC 60601-2-54 §203.7.8)
- **챔버 선택**: 3구역(좌/중앙/우) 독립 선택 또는 자동 선택 모드
- **Fail-safe**: AEC 센서 이상 시 촬영 자동 중단 + 알람 (필수 성능, IEC 60601-1 Clause 3.4)

### 3.3 자동 콜리메이션(Auto Collimation)
- **설계 요구**: Detector 활성 영역에 맞춰 X-ray 조사야 자동 제한
- **목적**: 불필요한 조직 피폭 제거, 산란선 감소
- **정확도**: Detector 유효 면적 대비 조사야 오차 ≤SID의 2% (IEC 60601-2-54)
- **수동 오버라이드**: 사용자가 수동으로 조사야 축소 가능, 확대 시 경고 표시

### 3.4 관전압·필터 자동 선택(APR: Anatomical Programmed Radiography)
- **설계 요구**: 검사 프로토콜(부위·체형) 선택 시 최적 kVp/mAs/필터 자동 설정
- **프로토콜 수**: 최소 60개 기본 프로토콜(성인/소아 × 부위)
- **소아 모드**: 체중/연령 기반 선량 감소 로직 (소아 DRL 참조)
- **편집**: 병원 방사선사가 프로토콜 파라미터 수정·저장 가능

### 3.5 선량 이력 관리 및 DRL 비교
- **설계 요구**: 환자별·프로토콜별 누적 선량 이력 저장
- **DRL 경고**: 검사별 DAP/입사선량이 국가 DRL 초과 시 촬영 전 팝업 경고
- **통계 대시보드**: 프로토콜별 평균/중앙값/75th percentile 선량 → DRL 대비 분석
- **데이터 출력**: RDSR(Radiation Dose Structured Report, DICOM) → Dose Management System 연계

### 3.6 그리드 자동 제거/삽입(Anti-Scatter Grid Control)
- **설계 요구**: 소아·사지 촬영 시 그리드 자동 제거 권장
- **목적**: 그리드 사용 불요 시 선량 30~50% 절감
- **인터페이스**: 프로토콜에 따라 그리드 사용 여부 기본값 설정, 사용자 오버라이드 가능

## 4. 위험 분석 연계 (ISO 14971)
| 위해 시나리오 | 심각도 | 발생확률 | 위험등급 | 경감 조치 |
|--------------|--------|---------|---------|-----------|
| AEC 실패 → 과노출 | S4 (중대) | P2 | 높음 | Fail-safe 차단 + 최대 mAs 하드웨어 리미트 |
| DAP 표시 부정확 → DRL 미인지 | S3 (보통) | P2 | 중간 | 정기 교정, 출하 시 ±25% 검증 |
| 자동콜리메이션 실패 → 과다 조사야 | S3 | P2 | 중간 | 수동 콜리메이션 백업 + 소프트웨어 자가진단 |
| APR 잘못된 프로토콜 적용 → 소아 과노출 | S4 | P1 | 중간 | 소아 모드 진입 시 확인 팝업 + 체중 입력 필수화 |
| DAP 기록 누락 → 선량 추적 불가 | S2 (경미) | P2 | 낮음 | RDSR 자동 생성, 실패 시 로그 알림 |

## 5. 설계 검증·확인 항목
| 항목 | 검증 방법 | 합격 기준 | 담당 |
|------|-----------|-----------|------|
| DAP 정확도 | 기준 DAP meter 대비 교정 | ±25% (IEC 60580) | QC 팀 |
| AEC 성능 | PMMA 팬텀 두께 변화 시험 | 선량 변동 ≤±25% | QC 팀 |
| 콜리메이션 정확도 | 규정 SID에서 필드 크기 측정 | ≤SID × 2% | 설계 팀 |
| DRL 경고 기능 | 시뮬레이션 데이터로 임계치 초과 | 경고 팝업 100% 발생 | SW 팀 |
| Fail-safe 동작 | AEC 센서 단선 시뮬레이션 | 촬영 중단 + 알람 100% | 설계+SW |
| RDSR 출력 | DICOM Validator로 구조 검증 | 필수 태그 100% 존재 | SW 팀 |
