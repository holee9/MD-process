## 1. FDA vs MDR vs MFDS — SBOM 요구사항 정합성

| 축 | 요구 시점 | 포맷 | 필수 항목 | 제출 방식 | 사후 갱신 |
|----|-----------|------|-----------|-----------|-----------|
| FDA (524B) | Premarket (2025-10-01 강제) | SPDX 2.3 / CycloneDX 1.5 | NTIA 7요소 + EOL + 취약점 | eSTAR 제출물 | Section 524B Postmarket Plan |
| MDR (MDCG 2019-16) | Technical Documentation Annex II | 포맷 권고 없음, IMDRF N73 원칙 | NTIA 7요소 + License + Hash | NB 심사 자료 | PMS Plan/PSUR 포함 |
| MFDS (2025-01-10) | 국내 허가 심사 | 권고 (SPDX/CycloneDX) | RA-07 항목 | 심사 자료 (권고) | 보안 지침 제16조 SBOM 관리 |

**정합성 판정: 본 SOP-SBOM-001 v0.1은 3축 모두 수용 가능한 상위 호환 설계**
- SPDX 2.3 또는 CycloneDX 1.5 채택 → FDA/MDR/MFDS 동시 만족
- NTIA Baseline 7요소 필수 → 공통 분모
- VEX 의무 채택 → FDA(권고→사실상 표준), MDR/MFDS(선택이나 업계 Best Practice)

## 2. 식별된 갭(GAP)

### G1. FDA 2025-10-01 시점까지 파일럿 SBOM 생성 실적 부재
- 현재 빌드 파이프라인에 SBOM 자동 생성 도구 미연동
- **조치**: 2026-Q3까지 Syft 또는 CycloneDX CLI 파일럿 통합, Q4에 정식 채택

### G2. 디지털의료제품법 사용적합성 자료 요건 미대응
- 기존 GSPR 체크리스트 v0.2에 사용적합성 증빙 컬럼 없음
- **조치**: GSPR 체크리스트 v0.3에 IEC 62366-1 Summative Evaluation 증빙 컬럼 추가

### G3. AI 모델 구성요소 단위 평가 제도(2026 시행)
- 본 프로젝트 AI 영상 분석 모듈 여부에 따라 재분류 필요
- **조치**: 2026-Q2 내 MFDS 구성요소 평가 세부 요건 확인, 적용 여부 결정

### G4. MFDS RA-01 ~ RA-20 체크리스트 전수 대응 문서화 미흡
- 현재 확보된 문서는 SBOM(RA-07) 중심
- **조치**: RA-01~RA-20 전체 매트릭스 작성 (별도 이슈)

### G5. 2025-07-18 진단용방사선 규칙(제1122호) 조문 원문 미확보
- 개정 주요 내용만 요약 — 형식시험 영향 판단 불가
- **조치**: 국가법령정보센터에서 원문 다운로드 후 v0.2 갱신

### G6. SOP-SBOM-001과 SOP-CC-001(Change Control) 인터페이스 미정의
- SBOM 변경이 Change Control 트리거하는 조건 명시 필요
- **조치**: SOP-CC-001 v0.2에 SBOM 컴포넌트 추가/제거/버전업 시 Change 분류 기준 반영

### G7. VEX 포맷 채택 결정(OpenVEX vs CycloneDX VEX)
- 현재 SOP에 양자 병기 — 운영 시 일관성 위해 단일 표준 선택 필요
- **조치**: Pilot 결과 바탕 2026-Q3 선택, v0.2 반영
