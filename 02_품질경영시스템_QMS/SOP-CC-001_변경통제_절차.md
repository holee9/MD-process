# SOP-CC-001 — 변경통제 절차 (Change Control Procedure) v0.1

문서번호: SOP-CC-001  
버전: v0.1 (초안)  
작성일: 2026-04-30  
작성자: 의료기기 업무규칙 개발팀  
관련 문서: ISO 13485:2016 §7.3.9, IEC 62304 §7, IEC 81001-5-1 §8, FDA QMSR (21 CFR 820 → ISO 13485 참조편입), EU MDR Art. 10(9)

---

## 1. 목적
의료용 X-ray 시스템/검출기/SW 제품의 설계·제조·품질에 영향을 미치는 변경을 체계적으로 관리하여, 안전성·유효성·규제 적합성을 유지한다. 특히 선량/영상품질(임상) 영향과 사이버보안 영향을 단일 Change Control Board(CCB)에서 병렬 판정하는 절차를 수립한다.

## 2. 적용 범위
- 설계 변경: HW(X-ray 튜브, Detector, 콜리메이터, 전원부), SW(펌웨어, 응용SW, AI 모델), 기계·외장
- 공정 변경: 조립 절차, 교정 절차, 시험 방법, 생산 환경·설비
- 문서 변경: QMS 절차서, 작업지시서, 라벨링, IFU
- 공급자 변경: 핵심 부품·원재료·SW 컴포넌트(OSS 포함) 변경
- 규제 변경 대응: 법규·표준 개정에 따른 적합성 재확인

## 3. 용어 정의

| 용어 | 정의 |
|------|------|
| CCB | Change Control Board — 변경 심의·승인 위원회 (설계, QA, RA, 보안, 생산 대표 포함) |
| CR | Change Request — 변경 요청서 |
| CIA | Change Impact Assessment — 변경 영향 평가 |
| Major Change | 안전성·유효성·의도된 사용에 영향을 미치는 변경 (규제 통지 가능성) |
| Minor Change | 문서 오류 수정, 외관 변경 등 안전성·유효성에 영향 없는 변경 |

## 4. 변경 분류 기준

### 4.1 분류 매트릭스

| 평가 항목 | Major | Minor |
|-----------|-------|-------|
| 환자 선량 영향 | 선량 ±10% 이상 변동 가능 | 선량 변동 없음 |
| 영상품질 영향 | MTF·DQE·균일성 사양 영향 | 영향 없음 |
| SW 안전성 클래스 변경 | IEC 62304 클래스 상향 또는 동급 핵심모듈 | 비안전 모듈, UI 외관만 |
| 사이버보안 영향 | 공격표면 변경, 암호알고리즘·인증방식 변경, SBOM 구성 변경 | 보안 무관 |
| 위험관리(ISO 14971) | 잔여위험 수용성 재평가 필요 | 기존 위험평가 유효 |
| 규제 적합성 | 허가사항 변경, GSPR 대응 변경, UDI DI 재발급 가능 | 규제 통지 불요 |

### 4.2 방사선 재교정 트리거 판정

아래 조건 중 하나라도 해당 시 **방사선 출력 재교정 필수** (Major 변경으로 분류):

| 트리거 조건 | 재교정 범위 | 근거 |
|------------|-----------|------|
| X-ray 튜브 교체 | kVp, mAs, HVL, 재현성, DAP 전항목 | IEC 60601-2-54 §201.12.1 |
| Detector 교체·펌웨어 업데이트 | Gain/Offset, 균일성, MTF, DQE, 잔상 | IEC 62220-1-1 |
| AEC 보드·센서 교체 | AEC 3챔버 감도, 선량 정확도 | IEC 60601-2-54 §203.7.8 |
| DAP 챔버 교체 | DAP 기준선량계 대비 검증 | IEC 60580:2019 §5 |
| 콜리메이터 교체·조정 | 조사야 정확도, 광조사야 일치도 | IEC 60601-2-54 §203.11 |
| 고전압 발생장치(HV Generator) 변경 | kVp, mAs, 파형 재현성, 선량 재현성 | IEC 60601-2-54 §201.12.1 |
| 방사선 차폐·필터 변경 | HVL, 누설선량, 선질 | IEC 60601-1-3 §7, §29.201 |
| SW 변경 — 선량 제어 로직 | AEC, APR, DAP 연산, 선량 표시 | IEC 60601-2-54 §203.7 |
| SW 변경 — 영상처리 파이프라인 | MTF, 균일성, 잡음 특성 | IEC 62220-1-1 |
| 전원부(인버터, 정류기) 교체 | kVp 파형, mAs 정확도, 재현성 | IEC 60601-2-54 §203.6 |

**재교정 후 합격 기준**: `04_제조공정_관리/공정_밸리데이션.md` §7.2 출하 교정 기준 준용  
**재교정 기록**: DHR에 편입, 변경 전후 비교 데이터 보존

## 5. 변경 프로세스

### 5.1 프로세스 흐름

```
[변경 식별] → [CR 작성·제출] → [접수·등록] → [영향 평가(CIA)]
    → [CCB 심의·승인/반려] → [구현·검증·확인] → [규제 통지 판정]
    → [문서 갱신·배포] → [CCB 종결 승인] → [기록 보존]
```

### 5.2 단계별 상세

#### (1) 변경 식별 및 CR 제출
- 변경 원인: CAPA, PMS 데이터, 규제 변경, 설계 개선, 공급자 변경, 고객 피드백
- CR 작성 항목: 변경 대상, 변경 사유, 긴급도(긴급/일반), 예상 영향 범위

#### (2) 영향 평가 (CIA)
병렬 평가 5축:

| 평가 축 | 평가자 | 산출물 |
|---------|-------|-------|
| DHF/설계 영향 | 설계팀 | 변경 도면/사양서 업데이트 범위 |
| RMF/위험관리 영향 | 위험관리자 | ISO 14971 위험평가 재수행 여부 |
| 사이버보안 영향 | 보안/SW팀 | Security Plan·SBOM 영향도, TARA 재수행 여부 |
| 방사선/선량 영향 | 방사선안전관리자 | 재교정 범위 (§4.2 트리거 참조) |
| 규제 적합성 영향 | RA | 규제 통지 유형 판정 |

#### (3) CCB 심의 및 승인
- CCB 구성: 설계, QA, RA, 보안, 생산 각 1인 이상 (방사선 관련 변경 시 방사선안전관리자 필수 참석)
- Minor: 서면 회의 가능 (이메일 승인)
- Major: 대면/화상 회의 필수, 회의록 보존
- 승인/조건부승인/반려 결정

#### (4) 구현·검증·확인
- 변경 구현 후 V&V 수행
- 방사선 재교정 해당 시: `04_제조공정_관리/공정_밸리데이션.md` §7.4 출하 판정 프로세스 준용
- SW 변경 시: IEC 62304 §8 문제해결 절차, 단위시험·통합시험·시스템시험 수행
- 보안 변경 시: IEC 81001-5-1 §8 보안패치 검증, SBOM 갱신
- AI 모델 변경 시: SOP-AIGOV-001 §5 성능평가·공정성·드리프트 재검증

#### (5) 규제 통지 의사결정

| 규제 기관 | Major 변경 시 통지 유형 | 판정 기준 |
|-----------|----------------------|-----------|
| MFDS | 변경허가 / 변경신고 | 의료기기법 시행규칙 제10조 |
| FDA | PMA Supplement / 510(k) 재제출 / Letter-to-File | FDA Guidance "Deciding When to Submit a 510(k)" |
| EU MDR | PCCP (Performance Change Control Plan) 또는 NB 통지 | MDR Art. 10(9), MDCG 2020-3 |
| UDI | DI 재발급 판정 | `06_문서_기록관리/SOP-UDI-001_UDI_통합관리_초안.md` §4.3 |

#### (6) 문서 갱신 및 종결
- 영향받는 문서 개정: SOP, WI, 도면, 라벨, IFU, 기술문서(TD), SBOM
- CCB 종결 승인 → CR 상태 Closed
- 변경 기록 보존: 제품 수명 + 2년 (ISO 13485 §4.2.5)

## 7. 긴급 변경 (Emergency Change)
- 환자·사용자 안전에 즉각적 위험이 있는 경우 CCB 소집 전 임시 조치 가능
- CCB 의장 + QA 책임자 사전 구두 승인 → 72시간 이내 정식 CR 소급 작성
- 긴급 변경도 V&V 필수 (단, 사후 수행 가능)
