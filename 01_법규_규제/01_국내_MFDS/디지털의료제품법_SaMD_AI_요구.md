# 디지털의료제품법 — SaMD/AI 의료기기 요구사항 정리 (2026-04 기준)

## 1. 법령 개요
- 법률명: 디지털의료제품법 (제20220호)
- 시행일: 2025-01-24
- 시행령·시행규칙: 2025-02-28 현행
- 소관: 식품의약품안전처(MFDS) 의료기기정책과 / 디지털헬스규제지원과
- 관련 고시: 디지털의료제품 허가·인증·신고·심사 및 평가 등에 관한 규정 / 분류 및 등급 지정 규정 / 디지털의료기기 제조 및 품질관리 기준 / 디지털의료기기 전자적 침해행위 보안 지침

## 2. 적용 대상
1. **디지털의료기기** — SW 단독(SaMD) 또는 SW 탑재 의료기기, AI/빅데이터·IoT·VR/AR 적용 제품
2. **디지털융합의약품** — 디지털기기와 결합된 의약품
3. **디지털건강지원기기** — 자율신고·자율성능인증 대상(의료기기 아님)

본 프로젝트 X-ray 시스템/검출기/SW 는 `디지털의료기기`에 해당될 가능성 높음(연결성·AI 적용 시).

## 3. 핵심 신규 요구사항
### 3.1 사용적합성(Usability)
- 허가 심사 시 사용적합성 평가 자료 제출 **의무화**
- IEC 62366-1:2015+A1:2020 기반, 독립형 SW는 별도 가이드라인(2025-01) 참조
- Summative Evaluation 결과 포함

### 3.2 사이버보안
- 사이버보안 요구사항 RA-01 ~ RA-20 체크리스트 제출
- 위협 모델(Threat Model), 위험평가, SBOM, 취약성 평가, 미해결 이상 평가, 추적성 통합 보고서
- 디지털의료기기 전자적 침해행위 보안 지침 제16조: SBOM 관리 활동 권고

### 3.3 변경관리 계획(PCCP 유사)
- AI 기반 디지털의료기기는 `변경관리계획` 사전 제출 가능
- 사전 승인된 범위 내 변경은 신속 반영(재심사 면제 또는 간소화)
- FDA PCCP / MDCG 2023-4 와 구조 유사

### 3.4 구성요소 단위 성능평가
- 완제품이 아닌 AI 모델 등 **구성요소 단위 성능평가** 제도 (2026 시행 예정)
- 플랫폼형 SW에서 모듈별 심사 경로 확보

### 3.5 자율신고·자율성능인증 (디지털건강지원기기)
- 웰니스·헬스케어 제품군은 의료기기 허가 대상 아님 — 별도 신고·자율인증
- 본 프로젝트는 해당 없음(의료기기)

## 4. 관련 가이드라인(2025년)
| 발행일 | 제목 | 관련성 |
|--------|------|--------|
| 2024-12 | 디지털치료기기 임상시험 설계 가이드라인 | 낮음(DTx 전용) |
| 2025-01 | 생성형 AI 의료기기 허가·심사 가이드라인 | 중(AI 기반 영상 분석 모듈 해당 시) |
| 2025-01 | 독립형 디지털의료기기SW 사용적합성 가이드라인 | 높음(뷰어·분석 SW) |
| 2025-01-10 | 의료기기 사이버보안 허가·심사 가이드라인(개정) | 높음 |
| 2025-01 | 디지털의료기기 전자적 침해행위 보안 지침(안) | 높음 |

## 5. 기존 의료기기법 대비 주요 변화
1. 기존 「의료기기법」에서 디지털의료제품 관련 규정을 분리·독립 입법
2. 사이버보안·사용적합성 요건 확대(요구 항목 15개 → 35개 수준)
3. AI/SW 제품 변경관리 계획 체계 신설
4. 완제품뿐 아니라 구성요소(모델) 단위 평가 가능
