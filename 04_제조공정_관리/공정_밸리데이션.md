# 제조공정 관리 및 밸리데이션 (v0.2)

> v0.2 — 2026-05-02: 절차 흐름도, IQ/OQ/PQ 양식, X-ray 시스템 적용예시, 판정기준 추가

## 1. 목적 및 적용 범위

### 1.1 목적
결과를 완제품 검사만으로 충분히 검증할 수 없는 제조공정에 대해 밸리데이션을 실시하고, 공정이 의도한 결과를 일관되게 달성함을 객관적 증거로 입증한다.

### 1.2 적용 범위
- 의료용 X-ray 시스템(촬영·투시), 디텍터, 영상처리 SW가 포함된 전 제품군
- 제조·조립·교정·시험·포장 공정 중 특수공정(Special Process)에 해당하는 것
- 생산·QMS에 사용하는 컴퓨터화 시스템(SW) 밸리데이션 포함

### 1.3 근거 규격
| 규격 | 조항 | 요구사항 |
|------|------|----------|
| ISO 13485:2016 | 7.5.6 | 공정 밸리데이션 — 결과를 후속 모니터링·측정만으로 검증 불가한 공정 |
| ISO 13485:2016 | 7.5.7 | 멸균 및 멸균 장벽 시스템 공정 밸리데이션 (해당 시) |
| ISO 13485:2016 | 7.6 | 모니터링·측정 장비 관리 |
| MFDS GMP 별표2 | 7.5 | 생산 및 서비스 제공 — 공정 밸리데이션 |
| FDA QMSR | ISO 13485 참조편입 | 기존 21 CFR 820.75 → ISO 13485 7.5.6으로 대체 |
| FDA | Process Validation Guidance (2011) | 3단계 접근: 공정 설계 → 공정 적격성확인 → 지속적 공정 검증 |

## 2. 밸리데이션 대상 공정 식별

### 2.1 특수공정 식별 기준
아래 조건 중 하나 이상 해당 시 특수공정으로 분류:
1. 공정 결과를 후속 검사·시험만으로 완전히 검증할 수 없는 경우
2. 파괴시험 없이는 적합성 확인이 불가능한 경우
3. 부적합이 사용 중에만 발견되는 경우

### 2.2 X-ray 시스템 특수공정 목록

| No. | 공정 | 특수공정 사유 | 밸리데이션 유형 |
|-----|------|--------------|----------------|
| SP-01 | Detector 모듈 접합(bonding) | 접합 강도를 비파괴로 100% 검증 불가 | IQ/OQ/PQ |
| SP-02 | X-ray 튜브 하우징 조립 (납 차폐) | 차폐 균일성을 외부 검사로 완전 검증 불가 | IQ/OQ/PQ |
| SP-03 | SW 이미지 캘리브레이션 프로세스 | 모든 촬영 조건 조합의 결과를 전수 검증 불가 | SW 밸리데이션 |
| SP-04 | 클린룸 조립 공정 (디텍터) | 파티클 오염을 사후 검사로 완전 검증 불가 | 환경 밸리데이션 |
| SP-05 | 생산용 SW 시스템 (MES, 교정 SW) | SW 자체가 품질에 직접 영향 | 컴퓨터화 시스템 밸리데이션 (CSV) |

## 4. IQ/OQ/PQ 단계별 상세

### 4.1 IQ (설치적격성확인)

| 항목 | 확인 내용 | 판정기준 | 기록 |
|------|-----------|----------|------|
| 장비 사양 일치 | 모델·시리얼·사양서 대비 | 사양서 100% 일치 | F-VAL-001 §IQ |
| 유틸리티 | 전원 전압·주파수·접지, 압축공기 압력 | 사양서 허용범위 내 | F-VAL-001 §IQ |
| 교정 상태 | 측정장비 교정 유효기간 확인 | 전 장비 유효기간 내 | 교정 성적서 사본 |
| 설치 환경 | 온도·습도·진동·청정도 | 장비 사양 및 클린룸 등급 충족 | 환경 모니터링 기록 |
| S/W 버전 | 설치된 SW 버전 대비 | 승인된 버전과 일치 | 스크린샷·해시값 |
| 도면·매뉴얼 | 최신 도면·매뉴얼 비치 | 최신판 확인 | 문서 목록 대조표 |

### 4.2 OQ (운전적격성확인)

| 항목 | 확인 내용 | 판정기준 | 기록 |
|------|-----------|----------|------|
| 공정 파라미터 범위 | CPP 상·하한 설정값 작동 | 설정값 ±허용공차 내 | F-VAL-001 §OQ |
| 최악 조건 시험 | 상한·하한 극한 조합 | 출력 CQA 합격 | 데이터시트 |
| 경보·인터록 | 이상 조건 시 경보 및 자동정지 | 100% 작동 | 시험 기록 |
| 반복성 | 동일 조건 반복 (최소 10회) | CoV ≤ 규정값 | 통계 분석표 |
| 전환 시험 | 공정 조건 변경 후 안정화 시간 | 규정 시간 이내 안정 | 시간 기록 |

### 4.3 PQ (성능적격성확인)

| 항목 | 확인 내용 | 판정기준 | 기록 |
|------|-----------|----------|------|
| 실생산 조건 반복 | 최소 3 Lot × Lot당 규정 수량 | 전 Lot CQA 합격 | F-VAL-001 §PQ |
| 통계적 검증 | Cpk 또는 합격률 산출 | Cpk ≥ 1.33 또는 합격률 100% | 통계 분석 보고서 |
| 작업자 변수 | 2인 이상 교대 실시 | 작업자 간 유의차 없음 | 분산분석표 |
| 공정 기록 완전성 | 배치기록서 항목 누락 없음 | 100% 기입 | 배치기록서 |

### 4.4 X-ray 디텍터 접합(Bonding) 공정 적용예시 (SP-01)

**CPP (중요 공정 파라미터):**
- 접합 온도: 150±5 °C
- 접합 압력: 2.0±0.2 MPa
- 접합 시간: 60±5 sec

**CQA (중요 품질 특성):**
- 접합 강도 (인장): ≥ 5 N/mm²
- 정렬 오차: ≤ 50 μm
- 결함 픽셀 수: ≤ 사양서 기준

**IQ**: 접합 장비 모델·온도 센서 교정 상태·치구 사양 확인
**OQ**: 온도 프로파일 매핑(9점), 압력 균일성, 최악 조건(145°C/155°C) 시험
**PQ**: 실생산 기판 3 Lot × 5개/Lot, 접합 강도·정렬·결함 픽셀 전수 검사

## 5. 소프트웨어 밸리데이션 (CSV)

### 5.1 대상
- 생산·QMS에 사용하는 S/W (MES, 교정 SW, LIMS, 문서관리 시스템)
- IEC 62304 (의료기기 S/W 수명주기) 대상은 설계개발 프로세스에서 별도 관리
- GAMP 5 카테고리 분류에 따른 위험기반 접근

### 5.2 GAMP 5 카테고리별 밸리데이션 수준

| 카테고리 | 설명 | 밸리데이션 수준 | 예시 |
|----------|------|----------------|------|
| Cat 1 | 인프라 SW (OS 등) | 설치·버전 기록 | Windows, Linux |
| Cat 3 | 상용 패키지 (설정 가능) | 설정 문서화·기능 시험 | ERP, Office |
| Cat 4 | 설정형 SW | 설정 검증·사용자 시험 | MES (설정 모듈) |
| Cat 5 | 맞춤 개발 SW | 전체 수명주기 밸리데이션 | 자체 교정 SW |

### 5.3 X-ray 교정 SW 밸리데이션 예시 (SP-05)
- **IQ**: SW 버전 확인, 실행 환경 사양 일치, DB 연결 확인
- **OQ**: 교정 알고리즘 정확도 (기준값 대비 ±허용공차), 에러 핸들링, 데이터 무결성
- **PQ**: 실 장비 3대 이상 교정 실행, 결과 재현성, 보고서 출력 정합성

## 6. 재밸리데이션 트리거

| 트리거 | 판단 기준 | 재밸리데이션 범위 |
|--------|-----------|------------------|
| 장비·설비 변경 | 주요 부품 교체, 이전 설치 | IQ부터 재실시 |
| 공정 파라미터 변경 | CPP 범위 변경 | OQ부터 재실시 |
| 원자재·부품 변경 | 공급자·등급·사양 변경 | OQ 또는 PQ |
| S/W 업데이트 | 공정 영향 SW 버전 변경 | 영향 범위에 따라 결정 |
| 부적합 경향 | SPC 관리도 이탈, 불량률 증가 | PQ 재실시 |
| 주기적 재검토 | 사업장 규정 (권장: 3년) | 전체 또는 PQ |
| 장기 미사용 | 6개월 이상 미가동 | OQ부터 재실시 |

재밸리데이션 결정은 변경통제 절차(SOP-CC-001)를 통해 문서화한다.

## 8. 방사선 장비 출하 교정 절차 연계

### 8.1 목적
진단용 방사선 안전관리규칙(제1122호) 정기검사 항목과 출하 전 교정·밸리데이션을 연계하여, 설치 후 검사 합격을 보증할 수 있는 출하 기준을 정의한다.

### 8.2 출하 교정 항목 및 연계 규격

| 출하 교정 항목 | 규칙 제1122호 검사항목 | IEC 규격 근거 | 교정 기준 | 교정 주기 |
|---------------|----------------------|--------------|-----------|-----------|
| 관전압(kVp) | 제5조 — 관전압 정확도 | IEC 60601-2-54 §201.12.1 | 설정값 대비 ±10% | 매 출하 |
| 조사시간(Timer) | 제5조 — 타이머 정확도 | IEC 60601-2-54 §203.6.3 | ±(10%+1ms) | 매 출하 |
| 선량 재현성 | 제5조 — 선량 재현성 | IEC 60601-2-54 §203.6.4 | CoV ≤ 0.05 | 매 출하 (5회 반복) |
| 반가층(HVL) | 제5조 — 선질 | IEC 60601-1-3 §7 | 관전압별 최소 HVL 충족 | 형식시험 + 변경 시 |
| 누설선량 | 제5조 — 누설선량 | IEC 60601-1-3 §29.201 | ≤1mGy/h @1m (100mA·min/h 기준) | 형식시험 + 변경 시 |
| DAP meter 교정 | 별표 — 선량표시 | IEC 60580:2019 §5 | 기준선량계 대비 ±25% | 매 출하 |
| AEC 교정 | 별표 — 자동노출제어 | IEC 60601-2-54 §203.7.8 | 두께 변화 시 선량 변동 ±25% | 매 출하 |
| 콜리메이션 정확도 | 별표 — 조사야 | IEC 60601-2-54 §203.11 | ≤SID × 2% | 매 출하 |
| 영상 해상도 | 별표 — 해상력 | IEC 62220-1-1 §5.4 | ≥설계 목표 MTF | 매 출하 |
| 균일성 | 별표 — 감도 균일성 | IEC 62220-1-1 §5.6 | ROI 편차 ≤15% | 매 출하 |

### 8.3 교정 장비 관리
- 기준선량계(Reference Dosimeter): KOLAS 인정 교정기관에서 연 1회 교정
- kVp 미터: 제조사 교정 또는 KOLAS 교정 (연 1회)
- 팬텀(PMMA, Line-pair, 저대비): 물리적 손상 점검 (연 1회), 교체 기준 문서화
- 교정 기록: QMS 기록으로 보존 (제품 수명 + 2년, ISO 13485 §4.2.5)

### 8.4 출하 판정 프로세스
1. 조립 완료 → 전기안전 사전시험(누설전류, 접지)
2. 방사선 출력 교정: kVp, mAs, Timer, HVL, 재현성
3. Detector 교정: 균일성 보정(Gain/Offset), 결함 픽셀 매핑
4. AEC 교정: 3챔버 감도 조정
5. 영상품질 확인: MTF, 균일성, 잔상, 저대비
6. DAP meter 교정: 기준선량계 대비 검증
7. 콜리메이션 정확도 확인
8. **출하 판정 체크시트 서명** (QC 담당 + 품질책임자)
9. 출하 교정 성적서 → DHR(Device History Record)에 편입

### 8.5 재교정 트리거
- X-ray 튜브 교체
- Detector 교체 또는 펌웨어 업데이트
- AEC 보드·센서 교체
- DAP 챔버 교체
- 고객 클레임(선량 이상, 영상품질 저하)
- 설치 후 설치검사(Acceptance Test) 시 기준 초과

### 8.6 설치검사와의 연계
- 출하 시 교정 성적서를 설치검사 시 기준 데이터로 사용
- 설치 환경(전원 변동, 접지 상태)에 의한 차이는 현장 미세 보정 후 기록
- 설치검사 합격 → 정기검사(3년 주기) 기산점

## 9. 투시(Fluoroscopy) 모드 출하시험 항목

> 4/30 교차검증 잔여 갭(G-30-02) 해소 — IEC 60601-2-54 §203.8, IEC 60601-2-43:2022, AAPM TG-272 기반.

### 9.1 적용 조건
- 본 항은 투시(Fluoroscopy) 기능이 포함된 장비(C-arm, R/F 겸용 등)에 한정하여 적용
- 정지영상(Radiography) 전용 장비는 §8 출하 교정 항목으로 갈음

### 9.2 투시 모드 출하시험 항목

### 9.3 시험 순서 및 주의사항
1. 장비 워밍업 완료 후 시험 개시 (최소 3분 이상)
2. FL-E01 → FL-E02 순서 엄수 (일반 → 고선량, 선량계 세팅 변경)
3. FL-E03 DAP 검증은 투시 모드 단독 실시 (정지영상 혼입 금지)
4. FL-E07 타이머 시험 시 최소 2회 반복하여 재현성 확인
5. 모든 결과를 출하 교정 성적서(DHR)에 기록, §8.4 #9와 통합 관리

### 9.4 불합격 시 조치
- 개별 항목 불합격 시 원인 분석 후 재시험 (최대 2회)
- 2회 재시험 불합격 시 CAPA 기록 후 설계·공정 검토
- FL-E01, FL-E02 불합격은 출하 차단 (환자 안전 직결)
